Follow Haller (2015) for naming and notation.

- create triplet or quadruplet particle sets (lon, lat resolution of the main grid, delta_lat, delta_lon for the probing displacements)
- create the flow map (final positions depending on initial positions) using parcels
- calculate derivatives based on the probing displacements
- $\nabla F$ based on the derivatives
- $C$ based on $\nabla F$
- eigen-value analysis of C
- FTLE based on the eigenvalues
- LCS based on tracing curves defined by eigenvectors of C
